# Launch_M&B2_by_Lelbry

GUI-лаунчер для **Mount & Blade II: Bannerlord** с балансными правками.

## Что умеет (план)

- [ ] Запускать Bannerlord с активным модом `LelbryBalanceFixes`
- [ ] Включать/выключать конкретные правки галочками
- [ ] Полный лут с врагов после битвы
- [ ] +50 к лимиту размера отряда по кнопке
- [ ] *(будет дополняться)*

## Стек

- Лаунчер: **C# / .NET 8 / Avalonia**
- Игровой мод: **C# + Harmony**, ставится в `Modules/LelbryBalanceFixes/`
